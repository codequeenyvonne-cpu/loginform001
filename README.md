## Overview

This project delivers a **modern login/signup interface** with a premium SaaS look:

- Glassmorphism form card
- Animated full-screen background slideshow
- Floating labels and inline icons
- Smooth micro-interactions and page transitions
- Client-side form validation with soft error feedback

## Modern UI Features

| Area | Details |
|---|---|
| **Visual Style** | Glass card, blur backdrop, gradient CTA button, modern typography |
| **Background Experience** | 5-image slideshow switching every 5 seconds with varied transitions |
| **Form UX** | Floating labels, icon-enhanced inputs, show/hide password toggle |
| **Feedback** | Animated error message, soft validation colors, shake on invalid submit |
| **Navigation** | Smooth transition between login and signup pages |
| **Accessibility** | Reduced-motion support for users with motion preferences |

---

## Tech Stack

- **HTML5** for semantic page structure
- **CSS3** for layout, glass effects, and keyframe animations
- **Vanilla JavaScript** for validation and UI behavior

No frameworks, no build step, no dependencies.
